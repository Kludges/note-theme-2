Note II

Theme for Nibbleblog 4.0

Download Fork on Github

Note II is a reload of its predecessor Note released on September 2012 for Nibbleblog 3.3 at the time.

The main feature of this theme has been its minimalist, clean and uncluttered design, with little or no support for plugins, targeting the end user who only needs/wants to maintain a basic personal journal, without redundant menus or functionalities.

What the new version brings

The new Note theme carries on the concept of minimal design and features, but this time i decided to add support for some basic utilities available by default with Nibbleblog like the new Pages functionality, Categories and About plugins, and support for Discus comments only.

The layout and color scheme were all redesigned and tweaked, fully responsive layout, using Roboto Condensed and Merriweather Serif from Google fonts. It works on IE9+, Firefox, Chrome, Safari, Opera 12+ and should work fine on all modern mobile browsers (not fully tested).

In depth

The Pages and Categories menus were neatly tucked into a hidden sidebar that can be accessed using the menu button next to the blog title, which toggles a css animation to slide in and out the sidebar. This functionality is only available if you choose to add those plugins, if not, no sidebar or menu button will be visible.

The main blog page (index) only shows the post titles and date published, keeping the tradition from the previous version, and supports pagination.

The post page shows large article title with date published and category, share links for tweeter, Google+ and facebook right after the article, along with the about plugin, if you choose to use it as an author card, and at last the Discus comment section - if activated.

Extra Stuff

This design was inspired by other blog apps, like medium.com, svbtle.com and roon.io, among others. After discussing some ideas with Diego Najar (creator of Nibbleblog), there were some features he liked from these other blog apps, like including individual large banner images for each post and the ability to navigate to next and previous post on post pages too.

These are going to be supported on the next Nibbleblog release (4.0.1) and are already being tested on the development repo on github, and so this theme will be updated to support these features as soon as they are available.

Matching Admin Theme

Just for the fun of it i decided to create a matching admin theme. If you decide to use it, download the .zip file, unpack, copy the folder named note-2-admin and drop it inside admin/templates/, then open the file admin/boot/rules/11-admin.bit in your favorite editor and change line 17 from
$default_template = 'easy4';
to
$default_template = 'note-2-admin';
Now login to your admin panel and enjoy!
